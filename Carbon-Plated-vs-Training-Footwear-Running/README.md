### Carbon-Plated vs. Training Footwear Running Analysis

This project compares running biomechanics between carbon-plated footwear and traditional training footwear using time-series gait data and discrete biomechanical variables.

**Focus Areas:**
- Carbon-plated footwear biomechanics
- Running gait analysis
- Lower-limb kinematics
- Ground reaction force interpretation
- SPM1D and paired t-test analysis

**Technical Stack:**
- Python / Google Colab
- pandas
- NumPy
- SciPy
- Matplotlib
- SPM1D

**Applied Value:**  
Demonstrates how footwear testing data can be cleaned, analyzed, and translated into product-performance insights.
